Server Authority 2021-08-21

I made the decision to make everything server authorative, however this brings up many questions. Ideally the client should see their resources going up by x amount / s, but how would this be server authorative? The server sending the resource data to the client every second would not be ideal. Maybe the client knows how much resource x / s they are getting just based off how much of whatever structure produces that resource, but wouldn't that go out of sync? Maybe there should be some data sent when the client purchases a new item that changes the rate at which they gain this resource?
